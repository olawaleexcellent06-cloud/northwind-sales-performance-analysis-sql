# 📊 Northwind Sales Performance Analysis (SQL)

## 📌 Project Overview
This project analyzes the Northwind database using MySQL to evaluate sales performance, customer purchasing behavior, and product revenue.
The analysis transforms transactional sales data into meaningful business insights by measuring revenue, identifying top-performing products, evaluating customer value, and tracking sales performance through SQL queries.

## 🎯 Business Problem

Businesses need to understand sales performance, customer purchasing behavior, and product profitability to make informed business decisions. Without proper analysis, it becomes difficult to identify high-value customers, top-performing products, and revenue trends.

This project uses SQL to answer key business questions such as:

- What is the total revenue generated?
- Which products generate the highest revenue?
- Which products sell the highest quantities?
- Who are the highest-value customers?
- What are the highest and lowest sales recorded?
- What is the average order value?

The analysis primarily uses:

### Orders
- Order ID
- Customer ID
- Order Date

### Order Details
- Product ID
- Unit Price
- Quantity

The relationship between these tables enables the analysis of sales performance, customer spending, and product revenue.

### 1. Data Exploration

### 2. Sales Performance Analysis

Calculated key performance metrics including:

- Total Revenue

### 3. Product Performance Analysis

Analyzed:

- Revenue generated by products
